# Todos-Application
A responsive Todo application built with HTML, CSS, and JavaScript that enables users to add, delete, and track tasks efficiently. It uses localStorage for data persistence, ensuring tasks remain across sessions, and demonstrates strong fundamentals in DOM manipulation, event handling, and clean UI design.
---

🌐 Live Demo

👉 https://saikumarbulasala.github.io/Todos-Application/

---

📌 Overview

This application ensures task persistence by leveraging the browser’s localStorage API, allowing all added tasks to remain intact even after page refreshes or browser restarts.

The project demonstrates strong fundamentals in frontend development, including DOM manipulation, event handling, and dynamic UI updates, while maintaining a smooth and responsive user experience.

---

✨ Features

- ➕ Add tasks dynamically
- ✅ Mark tasks as completed
- 🗑️ Delete tasks instantly
- 💾 Persist data using localStorage
- 🔄 Retain tasks after page refresh
- ⚡ Real-time UI updates without reload
- 🎯 Clean and responsive interface using Bootstrap
